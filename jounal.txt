コマンド･プロンプトから git コマンドでファイルを追加する.
    >git clone https://fanatoxin@github.com/fanatoxin/fanatoxin.github.io.git
    >cd fanatoxin.github.io
    >git config --global user.email fanayama@gmail.com
    >git config --global user.name fanatoxin
    ここで fanatoxin.github.io に index.html を作成する
    >git add index.html
    >git commit -m "#"
    >git push origin main
    info: please complete authentication in your browser...
    ここでログインを要求される
